MojiLMS is a full-stack Library Management System designed to simplify book management, user activity tracking, and administrative operations within libraries.
Developed using Flask, Jinja2, and SQLite, MojiLMS offers complete functionality for managing books, issuing and returning records, calculating fines, handling user accounts, and generating activity reports.
The platform includes secure OTP-based email registration, automated due-date reminders, and a clean, responsive interface suitable for all devices.

*Tech Stack*

Frontend: HTML5, CSS3, Bootstrap

Backend: Flask (Python), Jinja2

Database: SQLite

Authentication: Email OTP verification using Flask-Mail

Deployment: Render (or any Flask-supported hosting)


*Core Features*

*User Features* :

1.Account registration via secure OTP email verification

2.Session-based login authentication

3.Browse and search available books

4.Issue books with automatic due-date generation

5.Return books with fine calculation based on overdue days

6.View complete history of issued and returned books

7.Submit book requests directly to the administrator

8.Access daily, weekly, and monthly activity statistics

*Admin Features* :

1.Dedicated admin login

2.Add, edit, update, and delete books

3.Manage user accounts and handle requested books

4.Access library analytics, including:

  1.Total books available

  2.Total registered users

  3.Daily, weekly, and monthly issue statistics

  4.Most active members based on borrowing activity

*System Capabilities*

1.Automated email reminders for books nearing or past their due date

2.Contact form with email forwarding to the admin

3.Fully responsive UI using HTML and Bootstrap

4.Optimized queries ensuring smooth and fast application performance
