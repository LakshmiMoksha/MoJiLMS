MojiLMS is a full-stack Library Management System designed to simplify book management, user activity tracking, and administrative operations within libraries.
Developed using Flask, Jinja2, and SQLite, MojiLMS offers complete functionality for managing books, issuing and returning records, calculating fines, handling user accounts, and generating activity reports.
The platform includes secure OTP-based email registration, automated due-date reminders, and a clean, responsive interface suitable for all devices.

Tech Stack

Frontend: HTML5, CSS3, Bootstrap
Backend: Flask (Python), Jinja2
Database: SQLite
Authentication: Email OTP verification using Flask-Mail
Deployment: Render (or any Flask-supported hosting)

Core Features

User Features :

Account registration via secure OTP email verification

Session-based login authentication

Browse and search available books

Issue books with automatic due-date generation

Return books with fine calculation based on overdue days

View complete history of issued and returned books

Submit book requests directly to the administrator

Access daily, weekly, and monthly activity statistics

Admin Features :

Dedicated admin login

Add, edit, update, and delete books

Manage user accounts and handle requested books

Access library analytics, including:

Total books available

Total registered users

Daily, weekly, and monthly issue statistics

Most active members based on borrowing activity

System Capabilities :

Automated email reminders for books nearing or past their due date

Contact form with email forwarding to the admin

Fully responsive UI using HTML and Bootstrap

Optimized queries ensuring smooth and fast application performance
