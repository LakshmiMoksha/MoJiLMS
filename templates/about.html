About Our LMS Application

This Learning Management System (LMS) web application is designed to simplify and enhance online education. Built using Flask, HTML, CSS, and JavaScript, the platform offers a user-friendly interface for both students and instructors.

Key Features:

  • Course creation and enrollment
  • Secure user login and registration
  • Interactive content upload (videos, PDFs, assignments)
  • Quiz and evaluation modules
  • Dashboard to monitor performance

This project was developed as part of a web development learning journey, demonstrating both front-end design skills and back-end functionality. We aim to continue improving and adding new features to make learning more accessible and efficient.
